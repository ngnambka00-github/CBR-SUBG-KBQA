# CBR-SUBG-KBQA
This is the official implementation of the paper - [Knowledge Base Question Answering using Case-based Reasoning over Subgraphs](https://arxiv.org/abs/2202.10610).

## Installation
```bash
git clone ...
cd ...
pip env create -f environment.yml
```
